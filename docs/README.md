# agent-forge Documentation

Welcome to the comprehensive documentation of **agent-forge**, an advanced, experimental multi-agent operational ecosystem.

## Overview
This repository serves as a testing ground and sandbox for a decentralized society of highly specialized, persona-driven AI agents working in concert to design, orchestrate, audit, document, and test software systems and security infrastructures.

## Directory Structure
- `/agents/` - Decentralized agent nodes, containing their configurations, personalities, and localized knowledge.
- `/docs/` - Centralized documentation detailing architecture and agent capabilities.
- `/knowledge_base/` - The Great Library, containing the central archive of harvested agent knowledge and the Reaper job script.
- `/workflows/` - Standardized operational workflows for agent interactions and system modifications.
